# Step-by-step Guide: Deploying Your Website on AWS EC2 (Docker Compose)

## 1. Prepare Your Local Project
- Ensure your project is in a git repository or zipped for upload.

## 2. Create an AWS EC2 Instance
1. Go to https://aws.amazon.com/ and log in.
2. Click "Services" > "EC2".
3. Click "Launch Instance".
4. Name: personal-website
5. AMI: Ubuntu Server 22.04 LTS (HVM), SSD Volume Type
6. Instance type: t2.micro
7. Create new key pair (RSA, .pem), download and save it.
8. Network settings: Allow SSH, HTTP, and HTTPS from anywhere.
9. Click "Launch Instance".
10. Copy the Public IPv4 address.

## 3. Connect to EC2
- Open your terminal and run:


# On Windows (PowerShell):
# Skip the chmod step. Just run:
ssh -i <your-key-file.pem> ubuntu@<your-public-ip>

## 4. Update and Install Dependencies
sudo apt update
sudo apt upgrade -y
sudo apt install -y git docker.io docker-compose
sudo usermod -aG docker $USER
exit
# Reconnect:
ssh -i <your-key-file.pem> ubuntu@<your-public-ip>

## 5. Get Your Project on EC2
# Option A: Clone from GitHub

git clone <your-repo-url>.git
cd <your-repo-folder>

# Option B: Upload ZIP
# On your local machine:
scp -i <your-key-file.pem> <your-zipped-project>.zip ubuntu@<your-public-ip>:~/
# On EC2:
unzip <your-zipped-project>.zip
cd <your-project-folder>

## 6. Build and Run with Docker Compose
sudo docker-compose up --build -d

## 7. (Optional) Run Nginx with Custom Config
sudo docker run --name nginx -v $(pwd)/nginx.conf:/etc/nginx/nginx.conf:ro -p 80:80 -d nginx

## 8. Open Your Website
# In your browser:
http://<your-public-ip>

## 9. (Optional) Set Up Auto-Start on Reboot
sudo crontab -e
# Add this line:
@reboot cd /home/ubuntu/<your-project-folder> && sudo docker-compose up -d

# Done!
